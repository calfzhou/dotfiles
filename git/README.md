# Git Configurations

## Usage

### Mac and Linux

``` bash
cd ~
ln -s $DOTFILES_HOME/git/gitconfig .gitconfig
ln -s $DOTFILES_HOME/git/gitignore .gitignore_global
# In case if need some custom config:
touch .gitconfig_custom
```

cd %GIT_PROGRAM_PATH%
