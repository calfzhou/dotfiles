# Zsh (Oh-My-Zsh) Configurations

## Usage

```shell
cd ~
ln -s $DOTFILES_HOME/zsh/zsh-custom .zsh-custom
ln -s $DOTFILES_HOME/zsh/zshrc .zshrc
```

## Custom Scripts Loading Order

1. `__path.zsh`
2. `_custom.zsh` (if exists)
3. `aliases.zsh`
4. `example.zsh`
5. `exports.zsh`
6. `functions.zsh`
7. `~custom.zsh` (if exists)
